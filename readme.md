# Kira's API

Kira is my pet 🐕

![A Kira's pic](./images/chilling.jpg)

## Docs

### `/`

You will get a random image of Kira.

### `/docs`

You will be redirected to this page.

### `/:id`

You will get an specific Image of Kira.

### `/statistics`

You will get a list with all the images available. (JSON)
